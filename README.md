# NaijaPrice Tracker (Market-Oracle Protocol)

> Providing every Nigerian household with the **true cost of living** through real-time, location-aware commodity data.

---

## Overview

**NaijaPrice Tracker** is a frontend application built with **React (Vite)** and **TailwindCSS** that enables users to:

- Check real-time prices of commodities across Nigerian markets
- Compare prices by **State → LGA → Market**
- View price trends and daily changes
- Contribute price data (crowdsourcing model)

This project focuses on the **MVP "Utility Phase"** — helping users make informed buying decisions.

---

## Product Vision

To reduce the **Price Variance Gap** — the difference between what uninformed and informed buyers pay.

---

## Features (Frontend - MVP)

- **Localized Price Feed**
- Filter by location (State, LGA, Market)

- **Price Trends**
- View historical pricing (7-day, 30-day)

- **Price Indicators**
- Visual cues (↑ ↓) for daily changes

- **Add Price**
- Submit new commodity prices

- **Alerts (UI Ready)**
- Notification interface for price changes
